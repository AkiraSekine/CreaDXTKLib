Text



概要

文字列を描画するためのクラス
このクラスはシングルトンになっている

準備

  1. こちらにアクセスし、「MakeSpriteFont.exe」をダウンロードする。
  2. 保存した先でコマンドプロンプトを開く
  3. コマンドプロンプトで「MakeSpriteFont "(フォント名)" (保存ファイル名).spritefont /FontSize:(フォントサイズ)」を実行し、フォントファイルを作成する。

使用例

                    #include "CreaDXTKLib/Scene.h"
                    #include "Draw/Image.h"
                    #include "Math/Vector2.h"

                    using namespace CreaDXTKLib;
                    using namespace CreaDXTKLib::Draw;
                    using namespace CreaDXTKLib::Math;

                    // シーンクラスを基底としたシーンクラスの定義
                    class TextScene : public Scene
                    {
                    public:

                        // 更新処理
                        virtual void Update(float _elapsedTime) override
                        {
                            count++;
                        }

                        // 描画処理
                        virtual void OnRender() override
                        {
                            // countの値を描画
                            Text::Instance().Draw(L"Test", L"count=%d", count);
                        }

                    private:

                        int count = 0;
                    }

                    void SetUp()
                    {
                        // フォントファイルの読み込み
                        Text::Instance().Load(L"Test", L"font.spritefont");
                    }
