Singleton



概要

クラスをシングルトンにするためのクラス
テンプレートクラス
外部でのインスタンス、コピーコンストラクタ、代入演算は使用禁止

ヘッダファイル

Utility/Singleton.h

namespace

CreaDXTKLib::Utility

使用例

                    #include "Utility/Singleton.h"

                    using namespace CreaDXTKLib::Utility;

                    // シングルトンのクラスを定義
                    class SingletonClass : public Singleton<SingletonClass>
                    {
                        SINGLETON(SingletonClass)
                    }
