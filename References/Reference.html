イベント

宣言 概要
void(*EndFunction)() ゲーム終了時に呼ばれる関数ポインタ
void(*OnActive)() ウィンドウがアクティブ状態になった時に呼ばれる関数ポインタ
void(*OnInactive)() ウィンドウが非アクティブ状態になった時に呼ばれる関数ポインタ
void(*OnMinimize)() ウィンドウが最小化された時に呼ばれる関数ポインタ
void(*OnReturnMinimize)() ウィンドウが最小化から戻った時に呼ばれる関数ポインタ

使用例

#include "CreaDXTKLib/CreaDXTKLib.h"

CREA_DXTKLIB

int* hoge;

// 終了時処理
void GameEnd()
{
    delete hoge;
}

// エントリポイント
void SetUp()
{
    // 関数の設定
    EndFunction = GameEnd;

    hoge = new int;
}
